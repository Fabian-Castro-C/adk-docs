# Construir un agente de streaming

El Agent Development Kit (ADK) habilita experiencias interactivas en tiempo real con tus agentes de IA a través de streaming. Esto permite características como conversaciones de voz en vivo, uso de herramientas en tiempo real y actualizaciones continuas desde tu agente.

Esta página proporciona ejemplos de inicio rápido para que puedas comenzar a usar las capacidades de streaming tanto en Python como en Java ADK.
